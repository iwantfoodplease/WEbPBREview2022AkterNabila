Lab 8: Casscading Style Sheets

CSS stands for cascading style sheets. The css is used to style webpages and help layout the webpage. It is a style sheet language used to describe how HTML elements should be styled. CSS handles the color, font, size, adding aminations, decorative features, and spacing your content that you type in your webpage. There is a difference between HTML and CSS.HTML handles the structure of your webpage while CSS contains all the different features to make your web look appealing.
